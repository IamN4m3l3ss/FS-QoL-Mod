To install the mod simply install [melonloader](https://melonloader.co/) on FriendlyShooter and put the [dll](https://github.com/IamN4m3l3ss/FS-QoL-Mod/releases) in the mods folder.

To work on the code simply reference all .dll's in the dependency folder and it should build if it doesnt already.
