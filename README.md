To install the mod simply install [melonloader](https://melonwiki.xyz/#/) on FriendlyShooter and put the [dll](https://github.com/IamN4m3l3ss/FS-QoL-Mod/releases) in the mods folder.
For linux users add this to steam launch args: ```WINEDLLOVERRIDES="version=n,b" %command%```

To work on the code simply reference all .dll's in the dependency folder and it should build if it doesnt already.
